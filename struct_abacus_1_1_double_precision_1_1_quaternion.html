static void Abacus.DoublePrecision.Quaternion.Multiply ( ref Quaternion  q1,
ref Quaternion  q2,
out Quaternion  result 
)
inlinestatic

Performs muliplication of two Quaternion objects, (Quaternion multiplication is not commutative), (i^2 = j^2 = k^2 = i j k = -1).

For Quaternion division the notation q1 / q2 is not ideal, since Quaternion multiplication is not commutative we need to be able to distinguish between q1*(q2^-1) and (q2^-1)*q1. This is why Abacus does not have a division opperator. If you need a divide operation just multiply by the inverse.
